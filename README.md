# Fase_4 - Tech Challenge

Neste tech challenge, transformamos o modelo de séries temporais da IBOVESPA em uma aplicação iterativa com uso do Streamlit.

A aplicação utilizou o modelo desenvolvido na fase 2, XG Boost, que alcançou 77% de acurácia na previsão de subidas ou descidas o índice.

Nossa aplicação possui uma análise exploratória dos dados, onde o usuário pode selecionar o intervalo e verificar as métricas.

Em seguida, o usuário seleciona o período e tem a previsão de subida ou descida do índice.
